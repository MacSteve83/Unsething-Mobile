Supporto unsething mobile

unsething mobile consente di acquisire immagini, generare file DICOM e inviarli a un server DICOM/PACS configurato dall'utente.

Assistenza

Per assistenza, configurazione o segnalazione di problemi, contattare lo sviluppatore all'indirizzo email indicato nella pagina App Store dell'app.

Informazioni utili per il supporto

Quando si richiede assistenza, indicare il modello di iPhone o iPad, la versione iOS, il tipo di server DICOM/PACS utilizzato e una descrizione del problema riscontrato.
