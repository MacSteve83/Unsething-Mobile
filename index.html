Privacy Policy di unsething mobile

Ultimo aggiornamento: 11 maggio 2026

unsething mobile è un'app iOS che consente di acquisire immagini fotografiche, associare dati paziente, generare file DICOM e inviarli a un server DICOM/PACS configurato dall'utente.

Dati trattati dall'app

L'app può trattare localmente dati inseriti dall'utente, tra cui nome, cognome, identificativo paziente, data di nascita, sesso, descrizione dello studio e immagini acquisite tramite fotocamera o selezionate dall'utente.

Raccolta dei dati

Lo sviluppatore non raccoglie, non archivia e non vende dati personali, immagini o dati sanitari degli utenti tramite l'app.

Invio dei dati

I dati e le immagini possono essere convertiti in formato DICOM e inviati esclusivamente al server DICOM/PACS configurato dall'utente nelle impostazioni dell'app. La gestione, la sicurezza e la conservazione dei dati sul server di destinazione dipendono dall'infrastruttura scelta e configurata dall'utente.

Permessi richiesti

L'app può richiedere l'accesso alla fotocamera, alla libreria foto e alla rete locale per acquisire immagini e comunicare con il server DICOM/PACS configurato dall'utente.

Servizi di terze parti

L'app non usa sistemi pubblicitari, tracciamento di terze parti o strumenti di analytics integrati dallo sviluppatore.

Contatti

Per domande sulla privacy o per assistenza relativa all'app, contattare lo sviluppatore all'indirizzo email indicato nella pagina App Store dell'app.
