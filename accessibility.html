Accessibilita e permessi di unsething mobile

Ultimo aggiornamento: 11 maggio 2026

unsething mobile e' progettata per consentire l'acquisizione di immagini, l'inserimento dei dati paziente e l'invio dei file DICOM verso un server configurato dall'utente.

Permessi richiesti

L'app puo' richiedere l'accesso alla fotocamera per acquisire immagini cliniche e documenti, alla libreria foto per selezionare immagini gia' presenti sul dispositivo e alla rete locale per comunicare con il server DICOM/PACS impostato dall'utente.

Fotocamera e libreria foto

L'accesso alla fotocamera e alla libreria foto viene usato solo per acquisire o selezionare immagini da convertire in formato DICOM. Le immagini non vengono inviate allo sviluppatore.

Rete locale

L'accesso alla rete locale serve per eseguire il test C-ECHO e inviare immagini tramite C-STORE al server DICOM/PACS configurato dall'utente.

Accessibilita

L'app utilizza i controlli standard di iOS e supporta le impostazioni di sistema disponibili sul dispositivo. Per segnalazioni o necessita' specifiche di accessibilita', contattare lo sviluppatore tramite i recapiti indicati nella pagina App Store.
